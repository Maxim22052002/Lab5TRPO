Название:
1.Информационная система аптеки.

Содержание и цели:
2.Система предназначена для автоматизации учета продаж лекарственных препаратов и 
работы с поставщиками медикаментов с целью модернизации и повышения эффективности работы специалистов аптеки.

Аппаратные и программные требования:
System Requirement: OS: Windows 10, 8.1, 7 - 64bit Only
Processor: Intel i3 or equivalent
Memory: 2048 MB RAM  
Graphics: HD Graphics 4600 or above  
DirectX: Version 11  
Storage: 512 MB available space 

-Необходимо обеспечить программное взаимодействие системы с операционными системами Windows XP/7 и выше.
-Программа должна занимать не более 64 Мб оперативной памяти. Модули программы должны занимать не более 25 Мб памяти на жестком диске.

Инструкции по установке:

1)Выбор языка установки ПП.
2)Принятие/непринятие условий лицензионного соглашения.
3)Выбор папки установки.
4)Выберите папку в меня <<Пуск>>.
5)Выберите дополнительные задачи.
6)Всё готово, к установке.
7)Завершение установки и запуск ПП/выход из установщика.

Инструкции по эксплуатации:
1)Создание списка сотрудников, поставщиков и медикаментов.
2)Добавление новых сотрудников, поставщиков и медикаментов.
3)Регистрация продаж.
4)Просмотр продаж.
5)Составление заявки.
6)Просмотр остатка медикаментов.
